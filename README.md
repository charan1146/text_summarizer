# AI Text Summarizer

This project is a beginner-friendly AI Text Summarizer developed using Python, Streamlit, and NLTK.

The application takes a long paragraph as input and generates a short summary automatically using Natural Language Processing techniques.

## Features

- Summarizes long paragraphs into short text
- Simple and interactive Streamlit interface
- Uses NLP techniques for text processing
- Beginner-friendly implementation
- Fast and lightweight summarization

## Technologies Used

- Python
- Streamlit
- NLTK

## NLP Concepts Used

This project uses:

- Tokenization
- Stopword Removal
- Word Frequency Analysis
- Sentence Scoring
- Extractive Text Summarization
